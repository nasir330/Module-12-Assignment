# module_12_assignment

A simple Flutter calculator app with support for light/dark themes, operator handling, and backspace functionality.

## Features

- Basic arithmetic operations: `+`, `-`, `ร`, `รท`.
- Light and Dark mode toggle.
- Backspace button to delete last input.
- Prevents multiple operators in a row.
- Responsive layout using custom `CalcButton` widget.
